## 📋 Sobre o projeto

Este repositório documenta minha jornada de estudos em Python, com foco em lógica de programação. Ele é dividido em duas partes:

- **`app.py`** — Um sistema CLI de **cadastro e gerenciamento de restaurantes**, com menu interativo no terminal.
- **`exercicios1.py`, `exercicios2.py`, `exercicios3.py`** — Exercícios práticos cobrindo desde formatação de strings até estruturas condicionais, laços `for` e tratamento de exceções com `try/except`.

## ✅ Funcionalidades

### Sistema de Restaurantes (`app.py`)

- ✅ Menu interativo exibido no terminal, com arte ASCII de boas-vindas
- ✅ Cadastro de novos restaurantes, armazenados em lista
- ✅ Listagem de todos os restaurantes cadastrados
- ✅ Validação de opções inválidas com tratamento de erro (`try/except`)
- ✅ Navegação circular entre menu principal e submenus
- ✅ Limpeza de tela a cada nova tela (`os.system('cls')`)
- ⏳ Ativação de restaurante (funcionalidade em construção)

### Exercícios de lógica (`exercicios1.py` a `exercicios3.py`)

- ✅ Formatação de strings e f-strings
- ✅ Arredondamento de valores numéricos
- ✅ Estruturas condicionais (`if`, `elif`, `else`)
- ✅ Classificação de faixa etária e validação de usuário/senha
- ✅ Identificação de quadrante no plano cartesiano
- ✅ Criação e manipulação de listas
- ✅ Laços `for` (soma de ímpares, ordem decrescente, tabuada)
- ✅ Cálculo de soma e média com tratamento de exceções (`try/except`)

## 🛠️ Tecnologias

- **Python 3** — linguagem principal
- **Módulos nativos**: `os`, `statistics` (sem dependências externas)

## 📁 Estrutura do projeto

```
📦 repositorio
 ┣ 📜 app.py            → Sistema CLI de cadastro de restaurantes
 ┣ 📜 exercicios1.py     → Exercícios: strings, variáveis e formatação
```

## ▶️ Como executar

Certifique-se de ter o **Python 3** instalado. Depois, no terminal:

```bash
# Clone o repositório
git clone https://github.com/dida0982/nome-do-repositorio.git

# Acesse a pasta do projeto
cd nome-do-repositorio

# Execute o sistema de restaurantes
python app.py

# Ou execute qualquer arquivo de exercício
python exercicios1.py
```

> 💡 Os comandos de limpeza de tela (`os.system('cls')`) foram escritos para Windows. Em sistemas Linux/Mac, substitua por `os.system('clear')`.
